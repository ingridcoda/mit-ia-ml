# Fundamentos de Machine Learning com Scikit-Learn
**Projeto da Disciplina [Obrigatório]**
**Aluna:** Ingrid Ornellas Coda Sant'Anna Gomes

## Visão Geral
Este projeto aplica técnicas de Machine Learning Supervisionado para prever a inadimplência em cartões de crédito. O objetivo é construir, avaliar e comparar modelos, focando na utilidade prática para o negócio e respeitando a rubrica da disciplina.

## Como Executar
1. Instale as dependências: `pip install -r requirements.txt`
2. Certifique-se de que o arquivo de autenticação `kaggle.json` esteja na raiz do diretório.
3. Abra e execute o notebook: `IngridOrnellasCodaSantAnnaGomes_FundamentosdeMachineLearningcomScikitLearn_pd.ipynb`. Os dados serão baixados automaticamente para a pasta `data/`.
